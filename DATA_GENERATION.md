# Data Generation Notes

This repository uses synthetic data for local demos and phase testing.

## Demo Scenario

The phase 7 demo script `scripts/demo_narr05.py` builds a deterministic in-memory event store for the NARR-05 narrative:

- `ApplicationSubmitted` opens the loan request.
- `CreditAnalysisCompleted`, `FraudScreeningCompleted`, and `ComplianceCheckCompleted` provide supporting analysis.
- `DecisionGenerated` records the automated decline recommendation.
- `HumanReviewCompleted` records the manual override.
- `ApplicationApproved` captures the final underwritten decision.

## Outputs

The demo writes:

- `artifacts/regulatory_package_NARR05.json`
- `artifacts/narrative_test_results.txt`

The regulatory package is generated by `ledger.regulatory.generate_regulatory_package()` and includes:

- application summary data
- compliance state
- underwriting snapshot
- What-if counterfactuals
- a narrative summary

## Reproducibility

The scenario is deterministic and does not depend on external services. It can be regenerated locally by rerunning:

```bash
python scripts/demo_narr05.py --output-dir artifacts
```
